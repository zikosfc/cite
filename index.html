O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar e estruturar conteúdo na web.

Exemplo de código HTML básico:

<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    <h1>Meu Primeiro Título</h1>
    <p>Este é um parágrafo.</p>
</body>
</html>

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem para estilizar a apresentação de documentos HTML.

Exemplo de código CSS básico:

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

h1 {
    color: #333;
}

O que é JavaScript?

JavaScript é uma linguagem de programação que permite interatividade e dinamismo em páginas web.

Exemplo de código JavaScript básico:

document.getElementById('myButton').addEventListener('click', function() {
    alert('Botão clicado!');
});
