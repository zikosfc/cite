Como Usar HTML ``, `

` e CSS

Elemento `` - Títulos

O elemento `<h>` é usado para definir títulos e cabeçalhos.

Exemplo de Uso:

<h1>Título Principal</h1>
<h2>Subtítulo</h2>

Aplicando Estilos com CSS:

h1 {
    color: #333;
    font-size: 2.5em;
    /* Outras propriedades de estilo */
}

Elemento `

` - Parágrafos

O elemento `<p>` é usado para definir parágrafos de texto.

Exemplo de Uso:

<p>Este é um parágrafo de exemplo.</p>
<p>Outro parágrafo aqui.</p>

Estilização com CSS:

p {
    line-height: 1.6;
    margin-bottom: 1.5em;
    /* Outras propriedades de estilo */
}

CSS - Estilizando Elementos

O CSS (Cascading Style Sheets) é usado para estilizar elementos HTML.

Exemplo de Estilo:

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    /* Outras propriedades globais */
}
