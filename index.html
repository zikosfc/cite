Uso dos Títulos (h1-h6)

Os elementos de cabeçalho "h1" a "h6" são usados para definir títulos e subtítulos em uma página HTML.

Cada elemento de cabeçalho tem um nível de importância, sendo "h1" o mais importante e "h6" o menos. Eles são utilizados para estruturar o conteúdo da página de maneira hierárquica.

Exemplo:

Título Principal (h1)

Subtítulo (h2)

Sub-subtítulo (h3)

Sub-sub-subtítulo (h4)

Sub-sub-sub-subtítulo (h5)
Sub-sub-sub-sub-subtítulo (h6)

Uso de Parágrafos (p)

O elemento "p" é usado para representar um parágrafo de texto.

Ele é utilizado para agrupar blocos de texto em um parágrafo coeso e pode conter tanto texto simples quanto conteúdo mais complexo, como links e imagens.

Exemplo:

Este é um parágrafo simples.

Outro parágrafo para exemplificar.

Uso de Imagens (img)

O elemento "img" é usado para incluir imagens em uma página web.

Para exibir uma imagem, você precisa fornecer o caminho (`src`) para o arquivo de imagem. O atributo `alt` é usado para descrever a imagem para usuários que não conseguem visualizá-la.

Exemplo:

"<.img src="caminho-da-imagem.jpg" alt="Descrição da imagem".>"
